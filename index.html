# variables
name = Alice
write(Hello, @name!)

# list + loop
list.new(items)
list.add(items, apple)
list.add(items, banana)
list.len(items)
write(count: @_)

# offline AI
tinyai(Describe @name, r, .default, .default)
write(@r)

# browser
browser(chrome)
website(https://example.com)

# dialog

Installation

Up in 3 steps

Everything runs locally on Windows. No cloud account required.

1

Download the release

Grab the .zip from the download section below. Also grab the .vsix if you use VS Code for syntax highlighting.

2

Extract to a permanent folder

Unzip to a fixed location like C:\Mint\. The folder contains mint.exe, helper scripts, and the plugins\ directory.

3

Run setup.bat as Administrator

Right-click setup.bat → Run as administrator. It installs Python dependencies via mod.bat, registers .mint files in the Windows registry, and launches Mint.

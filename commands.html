Command Reference

Complete reference for the Mint scripting language (C build).
Commands are called as command(args). Args are comma-separated. Use @varname to inject variables anywhere.

01

Basic Commands

write( text )
output

Prints text or variable values to the console. Use @varname to inject a variable's value.

name = Alice
write(Hello, @name!)
write(Score: @score points)
ParameterTypeDescription
textText / VarContent to display. @var is expanded inline.
input( varname, prompt )
input

Displays a prompt and stores the user's response in a variable.

input(username, What is your name?)
write(Hello, @username!)
ParameterTypeDescription
varnameIDVariable name to store the result.
promptTextMessage shown to the user.
x = calc( expression )
math

Evaluates a math expression and stores the result. Supports + - * / %, parentheses, sqrt() abs() pow(x,y). Use @var inside to reference numbers.

a = calc(10 + 5)
b = calc(@a * 3)
c = calc(sqrt(144))
d = calc(pow(2, 8))
wait( seconds )
timing

Pauses execution for the given duration. Accepts decimals like 0.5.

write(Starting...)
wait(1.5)
write(Done!)
sys( command )
system

Passes a command directly to the OS shell.

sys(echo Hello from OS)
sys(mkdir output)
exit()
control

Immediately terminates the Mint process.

02

Control Flow

if( condition, { block } )
logic

Evaluates a condition. If true, runs the block. Operators: == != < > <= >= and logical and or not nand nor. Blocks can span multiple lines.

score = 85
if(@score >= 90, {
    write(Grade: A)
})
if(@score >= 75 and @score < 90, {
    write(Grade: B)
})
repeat( N, { block } )
loop

Runs the block exactly N times.

counter = 0
repeat(5, {
    counter = calc(@counter + 1)
    write(Count: @counter)
})
while( condition, { block } )
loop

Runs the block as long as the condition is true. Condition re-evaluated each iteration.

n = 1
while(@n < 100, {
    write(@n)
    n = calc(@n * 2)
})
func name { block }  ·  call name
function

Defines a reusable named function. call runs it. Functions share the global variable store.

func greet {
    write(Hello from a function!)
    write(Version: @version)
}
call greet
03

Lists

All list commands that return a value also store it in @_ for immediate use.
list.new( name )  ·  list.add( name, value )
list

Creates a new empty list. list.add appends a value to the end.

list.new(fruits)
list.add(fruits, apple)
list.add(fruits, banana)
list.add(fruits, cherry)
list.get( name, index )  ·  list.set( name, index, value )
list

list.get prints the item at that index and stores it in @_. list.set replaces it. Indexes start at 0.

list.get(fruits, 0)
write(First: @_)
list.set(fruits, 1, blueberry)
list.remove( name, index )  ·  list.pop( name )
list

list.remove deletes the item at an index. list.pop removes and returns the last item into @_.

list.remove(fruits, 0)
list.pop(fruits)
write(Popped: @_)
List Utilities
list
CommandDescription
list.len(name)Prints item count, stores in @_.
list.print(name)Prints all items numbered.
list.contains(name, value)Prints True / False, stores in @_.
list.indexof(name, value)Prints index or -1, stores in @_.
list.join(name, sep)Joins all items into a string, stores in @_.
list.reverse(name)Reverses the list in place.
list.clear(name)Empties the list.
list.delete(name)Destroys the list entirely.
list.join(fruits, |)
write(Joined: @_)
list.contains(fruits, apple)
write(Has apple: @_)
04

File Commands

run( file )
files

Executes a file. Supports .mint, .bat, and .reg.

run(setup.mint)
run(install.bat)
run(setup.reg)
ExtensionBehaviour
.mintInterpreted by Mint line by line.
.batExecuted via cmd /c (Windows).
.regApplied via reg import (requires admin).
file( path )
files

Opens or executes any file. Same extension support as run() plus executables.

file(notepad.exe)
file(launcher.bat)
locate( path, action )
files

Runs an action only if the given file exists.

locate(config.mint, run(config.mint))
05

Media

Handled by mint_media.py. Requires python-vlc, playsound3, climage.
image( file )
media

Renders an image in the terminal using Unicode block characters via climage.

image(photo.png)
video( file )
media

Plays video via VLC. Supports .mp4, .mkv, .avi and all VLC formats.

video(clip.mp4)
CommandDescription
vstop()Stop playback.
vvolume(N)Set volume 0–100.
vseek(ms)Jump to position in milliseconds.
vlength()Get total duration in milliseconds.
playsound( file )  ·  stopsound()
media

Plays an audio file non-blocking. Call stopsound() to stop it.

playsound(music.mp3)
wait(5)
stopsound()
06

Browser Automation

Handled by mint_browser.py. Requires selenium and ChromeDriver.
browser( mode )
browser

Opens a Chrome session. chrome for a visible window, incog for headless.

browser(chrome)
website(https://example.com)
click( BY, selector, wait )  ·  type( BY, selector, wait, text )
browser

Clicks or types into a page element. BY: ID, NAME, XPATH, CSS_SELECTOR.

click(ID, submit-btn, 10)
type(NAME, q, 10, hello world)
gettext( BY, selector, wait, varname )
browser

Reads text from a page element and stores it in a variable.

gettext(ID, result-box, 10, result)
write(Page said: @result)
07

UI Dialogs

Real tkinter popups via mint_dialog.py.
dialog( type, title, message [, varname] )
ui

Shows a native OS dialog. For ask / okcancel, the optional varname stores True or False.

dialog(info, Welcome, Mint is ready!)
dialog(ask, Confirm, Continue?, answer)
if(@answer == True, {
    write(User confirmed.)
})
TypeDescription
info / infInformation popup.
error / errError popup.
warning / warnWarning popup.
ask / yesnoYes/No — stores result in varname.
okcancel / confirmOK/Cancel — stores result in varname.
08

Artificial Intelligence

Handled by mint_ai.py. Install dependencies with mod.bat.
tinyai( prompt, varname, instructions, datafile )
ai · offline

Fully offline local LLM via a .gguf model and llama-server.exe. Server starts automatically.

tinyai(What is the capital of France?, reply, You are a geography expert., .default)
write(@reply)
ParameterTypeDescription
promptTextQuestion or instruction.
varnameIDVariable to store the AI response.
instructionsTextSystem prompt / role.
datafilePathContext file, or .default for dataset.txt.
ai( model, key, role, message, varname )
ai · online

Sends a prompt to an OpenAI-compatible API. Requires internet and a valid API key.

ai(gpt-4o, sk-YOUR-KEY, You are helpful., What is Mint?, reply)
write(@reply)
ParameterTypeDescription
modelStringModel ID, e.g. gpt-4o.
keyStringYour API key.
roleTextSystem role definition.
messageTextUser prompt.
varnameIDVariable to store the response.
llm( model, key, role, message, varname )
ai · multi-provider

Like ai() but routes through litellm, supporting Anthropic, Cohere, Mistral, and others.

llm(claude-3-5-haiku-20241022, sk-ant-KEY, You are helpful., Hello!, reply)
09

Plugins

Using Plugins
extend

Plugins extend Mint with new commands. Each plugin is a subfolder under plugins\ with a Plugin.py and Plugin.plug. They load at startup — no imports needed.

plugins\
  strtools\
    Plugin.py
    Plugin.plug

Each Plugin.plug registers commands:

script = plugins/strtools/Plugin.py
cmd    = upper
cmd    = lower
strtools plugin
built-in plugin

String utility plugin shipped with Mint.

CommandDescriptionExample
upper(text)Converts to UPPERCASEupper(hello) → HELLO
lower(text)Converts to lowercaselower(HELLO) → hello
reverse(text)Reverses the stringreverse(abc) → cba
length(text)Prints character countlength(Hello) → 5
replace(text, old, new)Swaps a substringreplace(hi world, world, Mint)
contains(text, search)Prints True or Falsecontains(Mint, int) → True
trim(text)Strips whitespacetrim( hi ) → hi
repeat_str(text, N)Repeats text N timesrepeat_str(ha, 3) → hahaha
